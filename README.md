# ✨ Glow Era — Beauty Brand Website

A fully functional, responsive website for **Glow Era**, a premium beauty and skincare brand. Built with vanilla HTML, CSS, and JavaScript.

## 📄 Pages

| Page | Description |
|------|-------------|
| **Home** (`index.html`) | Hero section, best sellers, about preview, testimonials, newsletter |
| **About** (`about.html`) | Brand story, values, team members |
| **Shop** (`shop.html`) | Product grid with category filters (All, Skincare, Makeup, Wellness, Gift Sets) |
| **Product** (`product.html`) | Single product detail with gallery, size/quantity selectors |
| **Blog** (`blog.html`) | Blog listing with article cards |
| **Blog Post** (`blog-post.html`) | Single blog article with full content layout |
| **Contact** (`contact.html`) | Contact form, business info, social links |
| **Cart** (`cart.html`) | Shopping cart with item list and order summary |

## 🎨 Design

- **Colors:** Warm rose, cream, gold — a soft, luxurious beauty palette
- **Typography:** Playfair Display (headings) + Inter (body)
- **Style:** Clean, modern, elegant with micro-interactions and hover states

## 🧩 Features

- **Responsive Design:** Mobile-first, adapts to tablet and desktop
- **Interactive JS:** Mobile nav toggle, product filters, quantity selector, cart counter, add-to-cart animations, scroll-triggered fade-in, form handling
- **Category Filtering:** Shop page filters products by category
- **Product Gallery:** Thumbnail navigation on product detail page
- **Newsletter & Contact Forms:** With submit animations

## 🚀 Getting Started

1. Open any `.html` file in a browser to view
2. No build tools or server required (static site)
